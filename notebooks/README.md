# Notebooks

This folder is intentionally included in the repository for reproducibility and review.

Included:

- training and experiment notebooks (`.ipynb`)
- small notes/results files

Not included:

- trained model weights (`.pt`, `.pth`, `.ckpt`, `.safetensors`)
- large generated artifacts that should live in `outputs/` or external storage

For this repository, trained weights are expected under `outputs/checkpoints/` during local runs and are ignored in Git.
